# Инструкция по подключению к VPN

Это руководство поможет вам подключиться к VPN для безопасного доступа к веб-приложению "Справочник телефонов ОмГТУ".

## Что такое VPN?

VPN (Virtual Private Network) - это виртуальная частная сеть, которая создает зашифрованное соединение между вашим устройством и удаленным сервером. Это обеспечивает:
- Безопасность данных
- Приватность в интернете
- Доступ к ресурсам, доступным только через VPN
- Защиту от отслеживания

## Популярные VPN-сервисы

### 1. OpenVPN (Рекомендуется для корпоративного использования)

**Установка:**
1. Скачайте клиент OpenVPN с официального сайта: https://openvpn.net/community-downloads/
2. Установите программу на ваш компьютер
3. Получите конфигурационные файлы (.ovpn) от администратора
4. Импортируйте файлы в клиент OpenVPN
5. Подключитесь к серверу

**Использование:**
- Запустите OpenVPN GUI
- Выберите нужный сервер из списка
- Нажмите "Connect"
- Введите логин и пароль (если требуется)

### 2. WireGuard (Современная альтернатива)

**Установка:**
1. Скачайте WireGuard с https://www.wireguard.com/install/
2. Установите клиент
3. Получите конфигурационный файл от администратора
4. Импортируйте конфигурацию
5. Активируйте соединение

**Преимущества:**
- Быстрее, чем OpenVPN
- Проще в настройке
- Меньше нагрузка на процессор

### 3. NordVPN / ExpressVPN (Коммерческие решения)

**Для личного использования:**
1. Зарегистрируйтесь на сайте провайдера
2. Скачайте приложение для вашей ОС
3. Войдите в аккаунт
4. Выберите сервер и подключитесь

## Настройка VPN на разных устройствах

### Windows

#### OpenVPN
1. Скачайте и установите OpenVPN Connect или OpenVPN GUI
2. Получите конфигурационный файл (.ovpn) от администратора
3. Импортируйте файл в приложение
4. Подключитесь

#### Встроенный VPN клиент Windows
1. Откройте "Параметры" → "Сеть и Интернет" → "VPN"
2. Нажмите "Добавить VPN-подключение"
3. Заполните данные:
   - Поставщик услуг: Windows (встроенный)
   - Имя подключения: ОмГТУ VPN (или любое другое)
   - Имя или адрес сервера: [получите у администратора]
   - Тип VPN: [уточните у администратора]
   - Имя пользователя и пароль: [получите у администратора]
4. Сохраните и подключитесь

### Android

1. Откройте "Настройки" → "Сеть и Интернет" → "VPN"
2. Нажмите "+" или "Добавить VPN"
3. Заполните данные:
   - Имя: ОмГТУ VPN
   - Тип: [уточните у администратора]
   - Адрес сервера: [получите у администратора]
   - Имя пользователя и пароль: [получите у администратора]
4. Сохраните
5. Нажмите на созданное подключение и введите пароль

**Или используйте приложение OpenVPN Connect:**
1. Установите из Google Play
2. Импортируйте .ovpn файл
3. Подключитесь

### iOS (iPhone/iPad)

1. Установите приложение OpenVPN Connect из App Store
2. Получите конфигурационный файл (.ovpn)
3. Откройте файл в приложении OpenVPN
4. Добавьте профиль
5. Подключитесь

**Или используйте встроенный VPN:**
1. Настройки → Основные → VPN и управление устройствами
2. Добавить конфигурацию VPN
3. Заполните данные и сохраните

### macOS

1. Системные настройки → Сеть
2. Нажмите "+" для добавления нового подключения
3. Выберите "VPN" в интерфейсе
4. Выберите тип VPN (L2TP, IKEv2, или другой)
5. Заполните данные сервера
6. Нажмите "Создать" и подключитесь

## Настройка VPN для доступа к приложению

После подключения к VPN:

1. **Убедитесь, что VPN активен**
   - Проверьте индикатор подключения в системе
   - Убедитесь, что вы можете получить доступ к внутренним ресурсам

2. **Откройте веб-приложение**
   - Используйте внутренний IP-адрес или домен, предоставленный администратором
   - Например: `http://192.168.1.100/oxirgi` или `http://internal.omgtu.local`

3. **Проверьте подключение**
   - Если страница не загружается, проверьте:
     - Активно ли VPN-соединение
     - Правильно ли указан адрес
     - Нет ли блокировок в файрволе

## Решение проблем

### Проблема: VPN не подключается

**Решения:**
1. Проверьте интернет-соединение
2. Убедитесь, что правильные учетные данные
3. Проверьте, не блокирует ли файрвол подключение
4. Попробуйте другой сервер VPN (если доступно)
5. Перезапустите VPN-клиент

### Проблема: Медленное соединение

**Решения:**
1. Выберите сервер ближе к вашему местоположению
2. Проверьте скорость интернета
3. Закройте ненужные приложения, использующие интернет
4. Попробуйте другой протокол VPN (если доступно)

### Проблема: Не могу получить доступ к приложению

**Решения:**
1. Убедитесь, что VPN подключен
2. Проверьте правильность адреса приложения
3. Попробуйте использовать IP-адрес вместо домена
4. Проверьте настройки файрвола
5. Обратитесь к администратору для проверки прав доступа

### Проблема: VPN отключается автоматически

**Решения:**
1. Проверьте настройки таймаута в VPN-клиенте
2. Убедитесь, что интернет-соединение стабильно
3. Проверьте настройки энергосбережения устройства
4. Обновите VPN-клиент до последней версии

## Безопасность

⚠️ **Важные рекомендации:**

1. **Используйте сильные пароли** для VPN-подключения
2. **Не делитесь** конфигурационными файлами с другими
3. **Отключайте VPN**, когда не используете
4. **Обновляйте** VPN-клиент регулярно
5. **Используйте двухфакторную аутентификацию** (если доступно)
