# Справочник телефонов ОмГТУ

Веб-приложение для управления контактной информацией сотрудников Омского государственного технического университета.

## Возможности

- Просмотр контактов сотрудников
- Поиск по имени, должности, отделу
- Управление подразделениями
- Админ-панель для управления данными
- Авторизация через Yandex

## Технологии

- PHP 7.4+
- MySQL
- Bootstrap 5
- Composer
